# HPP & Laba Rugi Calculator

Program sederhana untuk menghitung Harga Pokok Penjualan (HPP) dan laba/rugi berdasarkan data persediaan dan penjualan.

## Contoh Penggunaan
Masukan jumlah persediaan awal anda: 1000 Masukan jumlah pembelian bersih anda: 500 Masukan jumlah persediaan akhir anda: 300 Masukan total penjualan anda: 2000
HPP: 1200 Laba/Rugi: 800

## Fitur
- Menggunakan konsep Object-Oriented Programming (OOP)
- Input manual via terminal
- Output HPP dan laba/rugi secara otomatis

## Cara Menjalankan
```bash
python hpp_calculator.py
